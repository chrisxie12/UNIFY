# Framework & Project Structure

- **Framework**: Flutter 3.41.7 (Dart 3.11.5)
- **State Management**: Riverpod (flutter_riverpod)
- **Routing**: GoRouter (go_router) with StatefulShellRoute for bottom nav
- **Backend**: Supabase (supabase_flutter)
- **Fonts**: Google Fonts — Space Grotesk for ALL text
- **CSS**: No Tailwind. Custom Flutter theming with Material 3 ColorScheme + custom UColorScheme extension
- **Auth**: Supabase Auth (Google OAuth + email/password)
- **Notifications**: Local + FCM (graceful no-op)
