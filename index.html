The Problem

The Solution

How it works in 8 simple steps

1

Super Admin creates institution

Set up school profile and semester context.

2

Super Admin adds class

Create class spaces with secure codes.

3

Class Rep is assigned

Role-based permissions are granted.

4

Invite link is shared

5

Class Rep posts updates

Announcements, timetables, and tasks are published.

6

Students get notifications

Important updates arrive instantly.

7

Students set reminders

Private deadline and event reminders keep everyone on track.

8

Everyone sees edit history

Transparent change logs build trust and accountability.
