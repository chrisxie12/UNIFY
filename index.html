How Unify Works

  1. Super Admin registers a Class Rep and creates their class page.
  2. Class Rep activates the class page for 50 GHS per semester.
  3. Class Rep generates a one-time invite link and shares it via WhatsApp/SMS.
  4. Students join instantly using name/email.
  5. Class Rep posts announcements, assignments, and events with priority levels.
  6. Students receive real-time notifications for new or edited posts.
  7. Students set personal reminders to never miss deadlines.
  8. Edit history captures every post change, and everyone stays updated.
